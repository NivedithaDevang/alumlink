# 🎓 AlumLink

AlumLink is a web-based alumni networking platform that connects students and alumni through mentorship, networking, job opportunities, and community engagement. The platform enables meaningful interactions while helping institutions maintain strong alumni relationships.

## Features
- User Registration & Login
- Alumni & Student Profiles
- Alumni Directory
- Job & Internship Portal
- Mentorship Requests
- Events Management
- Community Networking
- Notifications & Messaging
- Admin Dashboard

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON / Database
- Git & GitHub
